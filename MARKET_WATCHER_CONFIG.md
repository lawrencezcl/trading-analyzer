# 15分钟市场观察任务配置

## 任务信息
- **任务名称:** market-watcher-15min
- **执行频率:** 每15分钟
- **执行时间:** 美股交易时段 (北京时间 周一到周五 21:30-06:00)
- **报告格式:** yyyymmddhhss.md

## 已配置定时任务
```cron
*/15 21-23,0-6 * * 1-5
```

## 报告保存位置
/root/.openclaw/workspace/market-reports/

## 报告内容
1. 实时价格数据 (SPX, NQ, JP225, DAX)
2. 威科夫分析 (吸筹/派发/上涨/下跌阶段)
3. 蔡森技术分析 (K线形态、支撑压力)
4. 交易计划 (方向、入场、停损、目标)
5. 市场整体评估
