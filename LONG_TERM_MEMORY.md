# 长期记忆系统
# Long-Term Memory System
# 创建于: 2026-02-22

## 记忆存储结构

### 1. 每日记忆 (memory/YYYY-MM-DD.md)
- 原始对话记录
- 当日事件
- 临时想法

### 2. 用户画像 (MEMORY.md)
- 基础信息
- 长期偏好
- 重要决策
- 性格特征

### 3. 知识库 (knowledge/)
- 用户专属知识
- 常用操作记录
- 项目历史
- 学习积累

### 4. 习惯模式 (patterns/)
- 作息规律
- 常用指令
- 偏好设置
- 行为模式

## 记忆策略

### 自动记录
- 每次对话自动追加到当日记忆
- 重要决策立即同步到 MEMORY.md
- 偏好变化实时更新

### 每日总结
- 每天自动总结当日对话
- 提取新的偏好和习惯
- 更新用户画像

### 增量学习
- 持续分析对话模式
- 自动调整响应风格
- 优化预测准确度

## 保护机制
- 记忆文件禁止删除
- 配置禁止重置
- 历史禁止清除
- 自动备份机制
