## 4.2 客户获取详细话术

### 4.2.1 初次拜访话术

**开场白**:
```
您好，我是[公司名]的[姓名]，我们专注于为本地制造企业提供AI数字化解决方案。

今天过来是想了解一下贵公司在数字化方面的情况，看看我们是否有合作的机会。

（递名片）
```

**需求挖掘**:
```
我想请教几个问题：

1. 目前贵公司的客户咨询主要通过什么渠道？
   - 电话？微信？网站？
   - 高峰期会不会出现响应不及时的情况？

2. 在客户服务方面，现在最大的困扰是什么？
   - 人力成本？响应速度？服务质量？

3. 有没有考虑过用AI技术来提升效率？
   - 对AI客服了解多少？
   - 有没有预算规划？

4. 目前使用的软件系统有哪些？
   - 是否需要集成对接？
```

**价值呈现**:
```
根据您刚才说的情况，我们的AI客服系统可以帮您：

1. **降低成本**
   - 一个AI客服可以替代3-4个人工客服
   - 月成本从2万降到1000元

2. **提升体验**
   - 秒级响应，不用等待
   - 7×24小时服务

3. **数据沉淀**
   - 自动记录所有对话
   - 分析客户需求，辅助决策

我们已经在[本地某企业]成功实施，他们的客服成本降低了70%，客户满意度反而提升了。

您看有没有兴趣详细了解一下？
```

**邀约下次**:
```
好的，感谢您的时间。

我这边准备了一个详细的方案，包含：
- 针对贵公司的解决方案
- 投入产出分析
- 实施计划

下周二或周三，我过来给您详细汇报一下，大概需要30分钟，您看哪个时间方便？
```

### 4.2.2 方案汇报话术

**开场**:
```
感谢[总/经理]抽出时间。

上次拜访后，我根据贵公司的情况，定制了一套AI数字化解决方案。

今天汇报三部分：
1. 贵公司现状分析和痛点
2. 我们的解决方案
3. 投入产出和实施计划

大概需要30分钟，我们直接开始？
```

**痛点共鸣**:
```
通过上次的沟通和我们的调研，发现贵公司目前主要面临这几个问题：

（根据前期调研的具体痛点展开）

这些问题，本质上都是可以通过数字化手段来解决的。
```

**方案介绍**:
```
我们的解决方案分为三个阶段：

**第一阶段: 快速见效（1个月）**
- 部署AI客服系统
- 解决80%的常见问题
- 预期效果: 客服成本降低50%

**第二阶段: 深度应用（2-3个月）**
- 集成订单系统
- 实现智能推荐
- 预期效果: 转化率提升20%

**第三阶段: 数据驱动（持续）**
- 数据分析看板
- 智能决策支持
- 预期效果: 运营效率全面提升
```

**报价与谈判**:
```
整个方案的投资如下：

- 系统部署费: X万元（一次性）
- 年度服务费: X万元/年
- 总计首年: X万元

这个投入，预计3-6个月可以通过降本增效收回。

（观察客户反应）

关于价格，考虑到我们是长期合作，我可以向公司申请：
- 首年服务费8折优惠
- 分期付款（首付50%，余款分6个月）

您看这样安排可以吗？
```

## 4.4 团队招聘指南

### 4.4.1 首年团队规划

```

├── 职责: 产品、技术、销售、管理
├── 人数: 1人

    │
    ├── 销售/运营助理 (第4个月招聘)
    │   ├── 职责: 客户开发、活动组织、内容运营
    │   ├── 要求: 本地人脉、沟通能力强
    │   ├── 薪资: 4000+提成
    │   └── 人数: 1人
    │
    └── 技术工程师 (第7个月招聘)
        ├── 职责: 产品开发、项目实施
        ├── 要求: 全栈开发能力
        ├── 薪资: 6000-8000
        └── 人数: 1人

外包合作:
├── UI设计: 按项目付费 (500-2000元/项目)
├── 内容创作: 按篇付费 (100-500元/篇)
└── 会计记账: 代理记账 (200-500元/月)
```

### 4.4.2 销售助理招聘JD

**岗位: 销售/运营助理**

**岗位职责**:
1. 负责客户开发，通过电话、拜访等方式获取客户线索
2. 组织线下活动，包括沙龙、培训等
3. 运营公司公众号、视频号等新媒体
4. 协助创始人进行商务谈判和合同签署
5. 维护客户关系，提高客户满意度

**任职要求**:
1. 大专及以上学历，1-3年工作经验
2. 有本地企业资源或销售经验优先
3. 沟通能力强，有亲和力
4. 学习能力强，对AI/数字化感兴趣
5. 有驾照，能独立开车拜访客户

**薪资福利**:
- 底薪: 4000元/月
- 提成: 签约金额的5-10%
- 综合收入: 6000-10000元/月
- 社保: 五险
- 工作时间: 弹性工作制

**面试问题**:
1. 介绍一下你之前的工作经历
2. 你对我们这个行业有什么了解？
3. 如果你来开发客户，你会怎么做？
4. 描述一次你成功说服别人的经历
5. 你期望的收入是多少？
